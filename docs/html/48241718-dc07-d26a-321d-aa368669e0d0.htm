UIElement Properties

The UIElement type exposes the following members.

Properties
  NameDescription
Container
The container of the UI element. For when UI elements need to exist within context menus, popup menus, etc.
Image
Image to draw to screen
Name
Internal name given to the UI element.
Pos
The position of the UI element. These coordinates are on screen. Where the UI element is a part of a UI container, the position is relative to the position of the container, otherwise is relative to the program window.
ScreenPos
The on-screen coordinates of the UI element
ScreenSize
The on-screen size of the UI element. Used when the container type is auto.
Size
The width and height of the UI element. Each component of the Point2D refers to the distance of that point from it's corresponding component in the Pos property.
